# 550-AU

Interactive real-physics visualisation of the Solar Gravitational Lens Telescope (SGLT) mission concept: solar-sail swarms flying to 650+ AU to use the Sun as a gravitational lens and image an exoplanet's surface. Named for the minimum solar gravitational focal distance, 547.8 AU.

Status: **planning complete, build starting.**

Open source (MIT). Repo: `m4cd4r4/550-AU`.

- `docs/SIM-PLAN.md` - full simulation plan (acts, physics model, scale strategy, assets, slices, verification)
- `docs/BUILD-PROMPT.md` - the build contract for the implementing session

Primary sources: Turyshev et al NIAC Phase III (arXiv:2002.11871), Turyshev & Andersson 2002 (arXiv:gr-qc/0205126), FOCAL (Maccone/Eshleman), NASA MMS formation-flying results, AstroKobi mission overview video (facts only, no script reuse).
